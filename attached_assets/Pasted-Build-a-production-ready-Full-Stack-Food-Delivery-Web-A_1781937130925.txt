Build a production-ready Full-Stack Food Delivery Web Application using Python Django, Django REST Framework, MySQL, HTML, CSS, JavaScript, Redis, Docker, Django Channels (WebSockets), and Razorpay Payment Gateway.

Project Name:
Food Delivery Management System with Online Payments, Real-Time Notifications, REST APIs, and Sales Analytics

Core Requirements:
Create a responsive, modern, secure, and scalable food delivery platform supporting Customers, Restaurant Owners, Delivery Partners, and Administrators.

Technology Stack:

* Backend: Python Django
* REST APIs: Django REST Framework
* Database: MySQL
* Frontend: HTML, CSS, JavaScript
* Authentication: JWT Authentication and Session Authentication
* Real-Time Communication: Django Channels + WebSockets
* Payment Gateway: Razorpay (Test Mode)
* Caching: Redis
* Containerization: Docker
* API Documentation: Swagger/OpenAPI
